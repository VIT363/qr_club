Закрытый клуб и решил сделать вход в него по одноразовому QR-коду. Приложив QR-код, в приложение приходит UUID участника. 
Если такой UUID присутствует в базе данных, то возвращается информация по участнику (ФИО) и UUID сменяется на новый.
Предусмотрите возможность добавления/изменения/удаления участников и qr кода через Rest контроллеры.

Вся необходимая информация по участникам должна хранится в базе данных. Используйте PostgreSQL. Базу данных нужно установить используя Docker.

Начальные данные в таблицах заполните вручную.
Приложение необходимо разрабатывать с использованием Spring Boot.
Проинициализировать проект можно используя онлайн инструмент.
По QR-кодом подразумевается тип данных UUID. В задаче необходимо создать 2 таблицы: participants (id, first_name, last_name, middle_name), 
qr_codes (id, participant_id, qr_uuid)
